Time of Day

A JavaScript Project by Noah Berhanu

Good Morning!

The html of this page changes color and the image changes based on the time of day using military time.

If the time is between 5:00-11:59, the html tag with an id of "my-greeting" will display "Good Morning!" on the page. The color will be yellow and image changes.

If the time is between 12:00-17:59, the html tag with an id of "my-greeting" will display "Good Afternoon!" on the page. The color will be lightblue and image changes.

If the time is between 18:00-22:59, the html tag with an id of "my-greeting" will display "Good Evening!" on the page. The color will be red and image changes.

Any other time (23:00-23:59 & 0:00-4:59), the html tag with an id of "my-greeting" will display "Good Night!" on the page. The color will be black and image changes.
